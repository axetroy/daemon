## daemon

`daemon` is a cli tool for run a process as daemon

### Usage

```bash
$ daemon [option] [command]
$ daemon --help
$ daemon --version
```

### Installation

If you are using Linux/macOS. you can install it with the following command:

```shell
curl -sf https://gobinaries.com/axetroy/daemon@v1.0.0 | sh
```
